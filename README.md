# DexAggregator

DexAggregator is an onchain entrypoint to execute calldata generated by smart order router algorithm running at the backend server.
User can make it to swap tokens among many liquidity sources in only one tx.

## Usage
```
pnpm install
# compile smart contracts
forge build # or pnpm hardhat compile

# test using forge
./tools/test.sh

# deploy to mainnet using hardhat or foundry
./tools/deploy.sh
```
